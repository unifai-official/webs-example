# פורטפוליו | אתרים ואפליקציות מובייל

דף כניסה המפצל לשני מסלולים: גלריית אתרים וגלריית אפליקציות מובייל.

## דפים
- `index.html` — דף כניסה: בחירת מסלול (אתרים / אפליקציות)
- `websites.html` — גלריית 18 אתרים עם סינון לפי קטגוריה ותצוגה חיה מלאה
- `apps.html` — גלריית אפליקציות מובייל בתוך פריים טלפון (4 אפליקציות)

## אפליקציות מובייל
האפליקציות בפורמט DC (`*.dc.html` + `support.js`) טוענות את ה-runtime ו-React מ-CDN
בעצמן, ולכן חייבות לרוץ דרך HTTP (לא `file://`). ב-Vercel ובכל שרת — הן עובדות.
במובייל הן נפתחות במסך מלא; בדסקטופ — בתוך פריים iPhone.

## פריסה ב-Vercel
אתר סטטי. `index.html` הוא דף הבית. `vercel.json` מפעיל `cleanUrls`.
ב-[vercel.com/new](https://vercel.com/new): Framework Preset = Other, ואז Deploy.

## בדיקה מקומית
האפליקציות דורשות שרת HTTP. הריצו בתיקייה:
`python -m http.server 8000` ואז גלשו ל-`http://localhost:8000`.

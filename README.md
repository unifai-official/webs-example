# פורטפוליו אתרים | Web Portfolio

דף פורטפוליו המציג אוסף אתרים שנבנו לעסקים מתחומים שונים — עורכי דין, רפואה, יופי, אוכל, חנויות, שירותים, בריאות ועיצוב.

## הדף הראשי
[`index.html`](index.html) — גלריית כל האתרים עם סינון לפי קטגוריה ותצוגה חיה מלאה של כל אתר.

## פריסה ב-Vercel
האתר סטטי (HTML/CSS/JS בלבד). הקובץ `vercel.json` מפנה את כתובת השורש `/` לדף הפורטפוליו.

1. ייבוא ה-repo ב-[vercel.com/new](https://vercel.com/new)
2. Framework Preset: **Other** (סטטי) — אין צורך ב-build
3. Deploy

עבודות נוספות בפיתוח שמורות כקבצי `.jsx` (React) ואינן נטענות בגלריה הסטטית.
